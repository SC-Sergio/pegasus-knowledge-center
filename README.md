# Pegasus Engineering Knowledge Center

Copiloto interno de documentación técnica para equipos de ingeniería, SRE y DevOps.

## Estado actual

- Aplicación base creada con Streamlit.
- Entorno local configurado con Python 3.12.
- Estructura inicial del proyecto creada.
- Próxima etapa: implementar lectura de documentos PDF.

## Ejecución local

``powershell
.\.venv\Scripts\Activate.ps1
streamlit run app\main.py
``

## Tecnologías iniciales

- Python
- Streamlit
- python-dotenv
